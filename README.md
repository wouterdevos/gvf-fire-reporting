# GVF Fire Reporting
A Go server that manages fire reports sent from the GVF Fire Reporting WhatsApp bot

## Configure environment variables
Create a `.env` file in the root directory and provide the following values:

```
VERIFY_TOKEN="your_chosen_verify_token"
ACCESS_TOKEN="your_permanent_meta_access_token"
PHONE_NUMBER_ID="your_whatsapp_phone_number_id"
PORT=8080
```
