# GVF Fire Reporting
A Go server that manages fire reports sent from the GVF Fire Reporting WhatsApp bot

## Configure environment variables
Create a `.env` file in the root directory and provide the following values:

```
VERIFY_TOKEN="webhook_verify_token"
ACCESS_TOKEN="permanent_meta_access_token"
PHONE_NUMBER_ID="whatsapp_business_phone_number_id"
PORT=8080
```
